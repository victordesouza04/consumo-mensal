Calculadora de consumo elétrico inteligente

Tem o objetivo de ajudar usuários a saber quanto de energia os aparelhos gastam por mês

Linguagem utilizada: Python

Formula utilizada para cálculo:
consumomensal = (potencia * tempo * 30) / 1000
gastomensal = (consumomensal * 0.75)

Para utilizar o programa você deve:
Informar qual é o nome do aparelho
Informar qual é a potência do aparelho em watts
Informar o tempo de uso diário em horas

O programa mostrará:
O nome do aparelho, seu consumo mensal em kWh e o gasto mensal estimado em R$
